Hak Akses Akun

Admin memiliki semua fitur. Owner dapat melihat seluruh rekapan semua cabang tanpa mengubah data. Tetapkan tepat satu toko untuk setiap akun Karyawan.

Buat akun baru melalui Supabase Dashboard → Authentication → Users. Akun pertama otomatis menjadi Admin; akun berikutnya menjadi Karyawan. Admin dapat mengubah akun menjadi Owner atau menetapkan satu toko untuk Karyawan. Pembatasan baca/tulis berlaku pada tampilan web dan Supabase Row Level Security.
