Hak Akses Akun

Admin memiliki semua fitur. Tetapkan tepat satu toko untuk setiap akun Karyawan; akun tersebut hanya dapat membuka dan memperbarui rekapan bahan toko pilihannya.

Buat akun baru melalui Supabase Dashboard → Authentication → Users. Akun pertama otomatis menjadi Admin; akun berikutnya menjadi Karyawan. Pilih peran Karyawan, pilih tepat satu toko, lalu tekan Simpan Akses. Pembatasan berlaku pada tampilan web dan Supabase Row Level Security.
