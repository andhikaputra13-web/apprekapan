Hak Akses Akun

Admin dapat mengatur akses akun. Owner dapat melihat akses akunnya sendiri dalam mode hanya lihat.

Buat akun baru melalui Supabase Dashboard → Authentication → Users. Admin dapat mengubah peran, memilih satu atau beberapa toko untuk Owner, dan memilih satu toko untuk Karyawan. Owner hanya dapat melihat akses akunnya sendiri. Pembatasan baca/tulis berlaku pada tampilan web dan Supabase Row Level Security.
