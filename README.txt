REKAPAN.ID - FILE PWA

Upload seluruh isi folder ini ke root project Vercel:
- index.html
- manifest.webmanifest
- sw.js
- folder icons/

Ikon kecil tab Chrome memakai icons/icon-16.png dan icon-32.png.
Ikon aplikasi memakai icon-192.png, icon-512.png, dan icon-maskable-512.png.
PWA/service worker hanya aktif melalui HTTPS atau localhost, bukan saat index.html dibuka langsung dengan file://.
